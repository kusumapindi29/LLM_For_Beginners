# LLM for Beginners

A practical learning repository for understanding modern LLM and GenAI systems through simple explanations, engineering intuition, and hands-on code examples.

This repository accompanies my Medium articles where I break down AI concepts step by step — from tokens and context windows to RAG systems, agents, and production AI patterns.

The goal is simple:
- understand concepts clearly
- build practical intuition
- implement small working examples
- learn AI systems from an engineering perspective
